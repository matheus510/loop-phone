### Getting started

Supported Node version: ```v8.5.0``` 

After cloning this repository, you will have to install all the dependencies:
```sh
$ npm install
```
To run the development enviroment (nodemon && babel):
```sh
$ npm run dev
```

Webpack and Docker are to be implemented.
